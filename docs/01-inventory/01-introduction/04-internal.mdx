## **Internal Netowrks** {#internal-apis}

Escape allows to deploy a Private Location and detects & fingerprint internal Application Assets behind firewalls or VPNs using a reverse tunnel.
This is the exact same feature that is used to scan Internal Applications using the Escape DAST.

Reference: [Escape Private Location Documentation](https://docs.escape.tech/platform/enterprise/repeater)
