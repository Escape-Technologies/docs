# Authenticated route bypass

## Description

Some data returned are not well advertised in your schema


## Remediation

Update your schema to match the real return types



## Configuration

> Identifier: `access_control/auth_bypass`



### Examples



#### Ignore this check

```json
{
    "checks": {
        "access_control/auth_bypass": {
            "skip": true
        }
    }
}
```
