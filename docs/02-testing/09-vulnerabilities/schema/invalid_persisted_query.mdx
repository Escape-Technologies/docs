# Invalid Persisted Query

## Description

In your schema, some persisted queries target non existant objects.

## Remediation

Make sure your graphql schema and persisted queries schema are in sync.


## Configuration

> Identifier: `schema/invalid_persisted_query`



### Examples



#### Ignore this check

```yaml
checks:
  schema/invalid_persisted_query:
    skip: true
```
